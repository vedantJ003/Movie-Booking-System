🔍 Scanner Image Integration Guide

📁 Where to Add Your Scanner Image

You can add your custom scanner image in several ways:

1. 📂 File Location

Place your scanner image in the public folder:

public/ ├── scanner-image.png ← Your custom scanner image ├── fingerprint-scanner.jpg ← Alternative name ├── my-scanner.png ← Another option └── ...

2. 🖼️ Image Requirements

  • Size: 120x120 pixels (recommended)
  • Format: PNG, JPG, or GIF
  • Content: Fingerprint scanner, biometric sensor, or similar
  • Style: Should look like a real scanner interface

🔧 How to Use Your Scanner Image

Method 1: Replace Default Image

Replace the default image file:

1. Save your image as "scanner-image.png" 2. Place it in the "public" folder 3. The scanner will automatically use your image

Method 2: Use JavaScript Functions

Use the built-in functions to set your image:

// Set a custom scanner image setScannerImage('your-scanner-image.png'); // Toggle between default and custom image toggleScannerImage();

Method 3: Direct HTML Modification

Edit the image source directly in the code:

<img src="your-scanner-image.png" alt="Fingerprint Scanner" class="scanner-image" id="scanner-image">

📝 Code Examples

Basic Usage

// Enable your custom scanner image setScannerImage('my-scanner.png'); // Toggle between default and custom toggleScannerImage();

Advanced Usage

// Set multiple scanner images setScannerImage('scanner-1.png'); // First image setScannerImage('scanner-2.png'); // Second image setScannerImage('scanner-3.png'); // Third image

Dynamic Image Loading

// Load image dynamically const imagePath = 'scanners/scanner-' + Math.floor(Math.random() * 3) + '.png'; setScannerImage(imagePath);
